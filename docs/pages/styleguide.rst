#############################
Style Guide for this project
#############################


RST is an odd markup language. In particular, there's no actual requirements for 
how to do headings. 

So its helpful to have some kind of style guide, even if the choices of style are arbitrary.

.. code-block:: rst

    ##################
    H1: document title
    ##################

    Introduction text.


    *********
    Sample H2
    *********

    Sample content.


    Sample H3
    =========

    Sample H4
    ---------

    Sample H5
